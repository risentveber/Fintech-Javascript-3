Сверстать полную страницу (без футера).

Макеты:

  • Desktop 1
  • Desktop 2
  • Mobile 1

Особенности:

  • Брейкпойнт для мобильной: 640px
  • До брейкпойнта верстка ломаться не должна, если ломается и никак иначе не избежать, ввести промежуточный брейкпойнт.
  • Использоваться БЭМ-нэйминг для классов.
  • Использовать less.
  • Повторяющиеся размера, цвета вынести в переменные.
  • Для элементов, у которых могут меняться состояние (цвет текста, бордер, фон, размер, позиция и тп), добавить transition.
  • Тултип также должен появляться с анимацией.
  • Для иконок использовать svg.
